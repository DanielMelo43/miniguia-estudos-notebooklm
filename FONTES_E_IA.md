# 📚 Metodologia e Referências de IA

Este documento detalha as fontes utilizadas e o papel do **Google NotebookLM** na construção deste caderno temático.

## 🛠️ O Papel do NotebookLM
Para garantir a precisão técnica e evitar alucinações comuns em modelos de linguagem generativa, utilizei o **NotebookLM** como motor de síntese. 

**Como a ferramenta foi utilizada:**
* **Processamento de Fontes:** As fontes listadas abaixo foram carregadas no ambiente para criar um "corpo de conhecimento" fechado.
* **Geração de Insights:** Os prompts documentados no README foram executados dentro deste contexto, garantindo que as definições de Python e Agentes seguissem estritamente a documentação oficial.
* **Estruturação de Dados:** A ferramenta auxiliou na organização dos "Mundos" do Python e na criação do glossário técnico.
